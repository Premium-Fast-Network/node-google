# Google SDK (NodeJS)

[![lerna](https://img.shields.io/badge/maintained%20with-lerna-cc00ff.svg)](https://lerna.js.org/)

Implement Google API For NodeJS With Separate Package

## Development

```
npm i -g typescript
npm i -g lerna
git clone https://github.com/Premium-Fast-Network/node-google
cd node-google
npm ci
lerna bootstrap
lerna run tsc
```
